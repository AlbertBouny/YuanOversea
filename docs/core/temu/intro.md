## 三、入驻要求
### 3.1 基础要求
| 项目 | 要求 | 说明 |
|-----|------|------|
| 主体资质 | 企业/工厂 | 营业执照、对公账户 |
| 品牌资质 | 品牌授权/自有品牌 | 商标证书、授权书 |
| 供应链 | 自主生产/稳定货源 | 产能证明、供应商 |
| 运营能力 | 专业团队 | 客服、物流、运营 |

### 3.2 类目要求
#### 重点类目
- 服装服饰
- 家居家装
- 美妆个护
- 3C数码
- 箱包配饰

#### 准入门槛
| 类目 | 要求 | 资质 |
|-----|------|------|
| 美妆个护 | 高 | 生产许可+品牌 |
| 3C数码 | 高 | 专利+认证 |
| 服装服饰 | 中 | 品牌授权 |
| 家居家装 | 中 | 品质认证 |
| 箱包配饰 | 低 | 基础资质 |
