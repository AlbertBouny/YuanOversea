# Temu平台规则指南

## 一、基础规则
### 1.1 店铺规则
| 项目 | 要求 | 处罚措施 |
|-----|------|---------|
| 发货时效 | 48小时内 | 扣分警告 |
| 客服响应 | 2小时内 | 考核扣分 |
| 商品质量 | 退货率≤3% | 限制上新 |
| 评价管理 | 好评≥4.8 | 流量限制 |

### 1.2 商品规则
#### 上架要求
- 主图规范
- 详情完整
- 价格合理
- 库存充足

#### 质量要求
- 品质达标
- 包装规范
- 标签完整
- 说明书齐全
