# TikTok视频运营指南

## 一、内容策略
### 1.1 内容定位
| 内容类型 | 特点 | 适用场景 |
|---------|------|---------|
| 产品展示 | 突出卖点 | 新品上新 |
| 使用教程 | 实用性强 | 复杂产品 |
| 场景应用 | 代入感强 | 生活用品 |
| 趣味测评 | 互动性高 | 爆款打造 |

### 1.2 内容规划
#### 内容矩阵
- 爆款视频：20%
- 种草视频：30%
- 品牌视频：20%
- 互动视频：30%

#### 发布频率
| 账号等级 | 发布频率 | 最佳时段 |
|---------|---------|---------|
| 新号 | 3-5条/天 | 12:00-14:00 |
| 成长号 | 5-8条/天 | 18:00-22:00 |
| 成熟号 | 8-12条/天 | 全时段覆盖 |

## 二、视频制作
### 2.1 拍摄规范
#### 基础要求
- 分辨率：1080P以上
- 比例：9:16
- 时长：15-60秒
- 帧率：≥30fps

#### 拍摄技巧
- 开头3秒抓眼球
- 画面稳定清晰
- 光线明亮自然
- 背景简洁干净
