# eBay平台介绍 Platform Introduction

最后更新时间：2024-03-21

## 一、平台概况
### 1.1 平台定位
- 全球最大C2C平台之一
- 拍卖+固价模式
- 二手和新品并存
- 小众品类优势

### 1.2 市场覆盖
- 主要市场：欧美、澳新
- 活跃买家：1.8亿+
- 在线商品：16亿+
- 190个国家市场

### 1.3 平台特点
- 类目齐全
- 买家保护政策
- 全球物流支持
- 多语言本地化
