## 常见问题

:::note FAQ
Q: 如何避免账号被警告或封禁？
A: 
1. 严格遵守平台规则
2. 定期检查账号健康度
3. 及时处理顾客反馈
4. 保持良好的绩效指标

Q: 收到警告该如何处理？
A:
1. 立即暂停相关操作
2. 分析违规原因
3. 准备申诉材料
4. 改进运营流程
:::
